# 🚀 Faqir's Portfolio

Welcome to **Faqir's Portfolio** — a clean, modern, and fully responsive personal website crafted to highlight my skills, projects, and creative identity.

**Live Demo:** [View Site](https://faqirullahafridi.github.io/Portfolio-new)  
**Source Code:** [GitHub Repository](https://github.com/faqirullahafridi/Portfolio-new)

---

## 🛠️ Technologies

- **HTML5:** Semantic, accessible markup
- **CSS3:** Flexbox, Grid, media queries, custom properties
- **JavaScript:** Vanilla JS (no frameworks)
- **SVG Icons:** Crisp, scalable graphics
- **PWA Ready:** Web manifest & favicons

---

## ✨ Features

- Minimal & modern UI
- Fast preloader animation
- Fully mobile-optimized
- Static contact form
- Built-in UI style guide
- Social media integration
- Flexible grid layout system
